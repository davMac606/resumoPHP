Datas em PHP

Em PHP, é possível criar e manipular datas. Existem vários comandos, os quais podem ser vistos na documentação de PHP

Para ser breve, mostrarei apenas as mais importantes(a.k.a q a Simone mostrou).

date()

Retorna a data atual do servidor.

Exemplo:

echo date("d/m/Y H:i:s");

Resultado:

time()

Retorna o timestamp atual do servidor.

Exemplo:

echo time();

Resultado:

mktime()

Retorna o timestamp de uma data específica.

Exemplo:

echo mktime(15, 30, 00, 02, 15, 2021);

Resultado:

strtotime()

Retorna o timestamp de uma data específica.

Exemplo:

echo strtotime("2021-02-15 15:30:00");

Resultado:

date_parse()

Retorna um array com as informações de uma data.

Exemplo:

print_r(date_parse("2021-02-15 15:30:00"));

Resultado:

date_parse_from_format()

Retorna um array com as informações de uma data, de acordo com um formato específico.

Exemplo:

print_r(date_parse_from_format("d/m/Y H:i:s", "15/02/2021 15:30:00"));

Resultado:

date_create()

Cria um objeto DateTime.

Exemplo:

$data = date_create("2021-02-15 15:30:00");

print_r($data);

Resultado:

date_format()

Formata uma data de acordo com um formato específico.

Exemplo:

$data = date_create("2021-02-15 15:30:00");

echo date_format($data, "d/m/Y H:i:s");

Resultado:

date_add()

Adiciona um intervalo de tempo a uma data.

Exemplo:

$data = date_create("2021-02-15 15:30:00");

date_add($data, date_interval_create_from_date_string("1 day"));

echo date_format($data, "d/m/Y H:i:s");

Resultado:

date_sub()

Subtrai um intervalo de tempo de uma data.

Exemplo:

$data = date_create("2021-02-15 15:30:00");

date_sub($data, date_interval_create_from_date_string("1 day"));

echo date_format($data, "d/m/Y H:i:s");
